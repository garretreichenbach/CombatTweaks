# Tactical Map

Press **Backslash** (configurable) to open a real-time tactical overlay of your current sector. The map shows all ships, stations, and asteroids with faction-colored indicators, distance readouts, and active order paths.

## Camera Controls

| Input | Action |
|-------|--------|
| **WASD** | Pan camera |
| **Space / Ctrl+Space** | Move camera up / down |
| **Shift** | Sprint (faster camera movement) |
| **Right-click drag** | Rotate camera |
| **Scroll wheel** | Adjust viewing range (100–20,000 units) |
| **X** | Reset camera to default position |
| **Double-click entity** | Focus camera on that entity |

## Selection

| Input | Action |
|-------|--------|
| **Left-click** | Select an entity (opens radial menu) |
| **Shift + click** | Add to current selection |
| **Click and drag** | Box-select entities in a rectangle |
| **Shift + drag** | Add box-selected entities to current selection |
| **Ctrl+A** | Select all friendly ships |
| **Ctrl+S** | Toggle turret selection mode |

### Turret Selection Mode

When turret selection mode is enabled (Ctrl+S), docked turret entities become visible on the tactical map with their own indicators. Clicking a ship in this mode opens the turret selector, allowing you to select individual turrets for targeting orders.

## HUD Elements

While the tactical map is open, context-sensitive keybind hints are displayed on the right side of the screen.

Two armor HP bars are available at all times (not just in tactical map mode):

- **Ship Armor HP Bar** — Vertical bar on the left side of the screen showing your current armor integrity percentage
- **Target Armor HP Bar** — Horizontal bar on the target panel showing the selected target's armor integrity
