# Thrust Blast

Thrust Blast is a quick burst of speed you trigger by **double-tapping a movement key** while piloting in flight — a fast dodge or closing burst without reaching for a separate ability key.

## Requirements

- The ship must have a **Thrust Blast** reactor chamber (the vanilla Mobility-tree chamber) installed and booted.
- You must be actively piloting the ship in flight mode.

## Using it

Double-tap any ship movement key — forward, backward, strafe left/right, or up/down — within a short window (about 300 ms). This fires the ship's built-in Take-Off (thrust blast) effect in that direction, the same effect the chamber normally provides, just bound to a double-tap so it's available mid-maneuver.

If the ship has no Thrust Blast chamber, the double-tap does nothing.

## Notes

- Detection is client-side; the activation is validated and applied on the server, so it respects the chamber's normal charge and cooldown.
- The Thrust Blast chamber's strength upgrades (vanilla) still govern how strong the burst is.
