# Installation

## Requirements

- StarMade with StarLoader/StarMade-Open mod support

## Installing the Mod

1. Download the latest release JAR from the [GitHub releases page](https://github.com/garretreichenbach/CombatTweaks/releases)
2. Place it in your StarMade `mods/` directory
3. Launch StarMade — the mod loads automatically

The mod will generate its configuration files on first run. See [Configuration](configuration.md) for details.
