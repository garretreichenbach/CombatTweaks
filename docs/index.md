# CombatTweaks

A StarMade mod that adds real-time tactical fleet management and an armor HP defense system. Command multiple ships through an RTS-style tactical map overlay, issue fleet orders, and benefit from a passive armor durability layer that rewards building with armor blocks.

## Overview

CombatTweaks brings RTS-style fleet command to StarMade. Open the tactical map to get a bird's-eye view of your sector, select ships, and issue orders — all without leaving your cockpit.

### Key Features

- **Tactical Map** — Real-time sector overlay showing all ships, stations, and asteroids with distance readouts and order paths
- **Fleet AI Orders** — Command ships to attack, defend, move, mine, or repair through a radial menu
- **Armor HP System** — Passive armor durability layer that absorbs damage before it reaches your reactor
- **HUD Elements** — Armor integrity bars for your ship and your current target

## Quick Start

1. [Install the mod](getting-started/installation.md)
2. Enter a ship and press **Backslash** to open the tactical map
3. Click and drag to select ships, then click a target to issue orders

For configuration options, see the [Configuration](getting-started/configuration.md) page.
